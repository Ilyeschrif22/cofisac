COFISAC corporate website built with React and Vite.

## Project Structure

- `src/components` - reusable UI sections
- `src/pages` - page-level views
- `src/routing` - custom client-side routing
- `public` - static assets such as images and video

## Available Scripts

```bash
npm run dev
npm run build
npm run preview
```

## Pages

- `/` - Home
- `/qui-sommes-nous` - Qui sommes nous
- `/contact` - Contact

## Deployment

The project includes `vercel.json` so direct navigation and page refreshes work on Vercel.

## Notes

- The site uses a custom router instead of React Router.
- Static assets are served from `public`.
